# BOOTH POS

행사 부스용 반응형 웹 POS입니다.

## 사이트

https://xxoxx1309-byte.github.io/booth-pos/

## 실행

```bash
npm install
npm run dev
```

같은 Wi-Fi의 스마트폰이나 태블릿에서는 터미널에 표시되는 `Network` 주소로 접속할 수 있습니다.

## 포함 기능

- 상품 카탈로그, 검색, 카테고리 필터
- 실시간 수량 및 주문 합계
- 3종 이상 구매 시 자동 세트 할인, 수동 할인
- 받은 금액 및 거스름돈 계산
- 판매 완료, 재고 차감, 거래 초기화
- 버전 관리되는 브라우저 `IndexedDB` 기반 판매 기록 및 재고
- 기존 `localStorage` 데이터 자동 이전
- JSON 백업 다운로드, 복원 및 전체 초기화
- 큰 글자, 고대비, 키보드 포커스, 스크린리더 안내
- 모바일 장바구니 하단 시트와 반응형 내비게이션

## 데이터 보관

판매 기록, 재고와 접근성 설정은 현재 기기의 브라우저 `IndexedDB`에 저장됩니다.
설정 화면의 **백업 다운로드**를 사용하면 다른 기기에서 복원할 수 있는 JSON 파일이 생성됩니다.
브라우저 앱 데이터 삭제나 시크릿 모드에서는 로컬 데이터가 사라질 수 있으므로 행사 전후로 백업하는 것을 권장합니다.
